1. SECTION IDs — NAV TARGETS
═══════════════════════════════════════════════════════════════════

Each nav link routes to one of these section IDs.
The breaker section always holds the ID; the gallery/cards that
follow are the same visual chapter but have no ID.

  #hero          → hero section (top of page)
  #philosophy    → about / philosophy + stats section
  #how-i-work    → how I work tile grid
  #work          → MADE REAL breaker (dark, full-viewport)
                   followed immediately by work cards (no id)
  #object        → Object breaker (light, full-viewport)
                   followed immediately by object gallery (no id)
  #contact       → contact + Ask AI section

6. SCROLL REVEAL SYSTEM
═══════════════════════════════════════════════════════════════════

Add to any element to animate it in on scroll:

  .r        Base class. opacity:0, translateY(22px). JS adds .on
            when element enters viewport → opacity:1, transform:none.

  .d1       Delay 0.07s
  .d2       Delay 0.14s
  .d3       Delay 0.21s
  .d4       Delay 0.28s
  .d5       Delay 0.35s
  .d6       Delay 0.42s

Usage: <div class="r d2"> animates in with a 0.14s delay.
Combine multiple elements with ascending delays for stagger effect.

Transition duration: --dur-reveal (0.95s), easing: ease.

9. CANVAS DOT RIPPLE (interactive)
═══════════════════════════════════════════════════════════════════

Cursor pushes nearby dots away; they lerp back when cursor moves.
Applied to: .contact-zone, .philosophy-sec#work (MADE REAL).

HTML required inside the section:
  <canvas class="cz-canvas" aria-hidden="true"></canvas>
  <div class="cz-dots-static" aria-hidden="true"></div>

JS: initDotRipple(element) — defined once, called per section:
  initDotRipple(document.querySelector('.contact-zone'));
  initDotRipple(document.getElementById('work'));

Parameters (currently hardcoded in JS):
  SPACING = 32px grid
  DOT_R   = 1.5px radius
  DOT_COLOR = rgba(255,255,255,0.10)
  RIPPLE_RADIUS = 150px
  MAX_DISPLACE  = 18px
  LERP          = 0.12 (snap-back speed)

Mobile (pointer:coarse): canvas hidden, .cz-dots-static shown.
Desktop: section backgroundImage set to 'none' (canvas replaces it).

CSS:
  .cz-canvas      position:absolute; inset:0; z-index:0.
  .cz-dots-static position:absolute; inset:0; display:none by default.

  SVG rect with pathLength="1000". On hover: stroke-dashoffset

  Easing: cubic-bezier(0.4, 0, 0.2, 1), duration 0.55s.

  HTML for one dot arm:

  Keyframes: hiw-spin-cw, hiw-spin-ccw

═══════════════════════════════════════════════════════════════════
15. INTERACTION PATTERNS
═══════════════════════════════════════════════════════════════════

CANVAS DOT RIPPLE
  Cursor displaces nearby dots (within 150px radius).
  Math: displacement = (radius - dist) / radius * maxDisplace
  Dots lerp back: offset += (target - offset) * 0.12
  Implemented in initDotRipple(el). Pass any section element.

TWO-FACE CARD (how-i-work tiles)
  Default face fades out + translates up (-6px).
  Hover face fades in + translates up from 10px offset, 0.1s delay.
  Fixed card height (260px) — no layout shift ever.
  Title is duplicated in both faces (one per face in HTML).

SVG BORDER TRACE
  SVG <rect> with pathLength="1000" normalization.
  stroke-dashoffset: 1000 → 0 on hover.
  Smooth ease-in-out cubic-bezier, no bounce.

COPY TO CLIPBOARD
  .cz-copy-btn click → navigator.clipboard.writeText()
  Adds .copied class for 2000ms → icon swaps copy → check.
  Fallback: textarea + execCommand for older browsers.

SCROLL-SPY NAV
  IntersectionObserver watches all section[id] elements.
  Active link class applied to nav <a> matching section id.
  Threshold: whichever section is most visible gets highlighted.

SCROLL REVEAL
  IntersectionObserver watches all .r elements.
  Adds .on class when element enters viewport.
  Stagger via .d1–.d6 delay classes.
